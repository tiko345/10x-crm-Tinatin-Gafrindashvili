# CRM Client Management System

A simple, client-side CRM (Client Relationship Management) system built with HTML, CSS, and JavaScript. This project allows you to add, view, filter, sort, and manage clients, including their details, statuses, and notes.

[App Live Demo](https://10x-crm-tinatin-gafrindashvili.vercel.app)

##  Features
- Add Clients: Add new clients with details like name, email, phone, company, and deal value.
- View Clients: Display clients in a card-based layout.
- Filter Clients: Filter clients by status (Lead, Contacted, Won, Lost).
- Search Clients: Search clients by name or company in real-time.
- Sort Clients: Sort clients by name, deal value, or creation date.
- Edit Client Status: Change the status of a client directly from the card.
- Delete Clients: Remove clients with a confirmation prompt.
- Client Details: View detailed information about a client, including notes.
- Add Notes: Add and view notes for each client.
- Set Reminders: Set a reminder to follow up with a client (toast notification).
- Persistent Data: Client data is saved to `localStorage` and persists across page refreshes.



## Technologies Used
- Frontend: HTML5, CSS3, JavaScript 
- Storage: `localStorage` (for data persistence)
- API: [DummyJSON](https://dummyjson.com/) (for mock data)
- Styling: Custom SCSS/CSS (no external libraries)
